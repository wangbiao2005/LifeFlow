
# LifeFlow 本地兜底数据文档

当 AI 服务不可用（无 Key 或网络错误）时，系统将使用以下本地预设数据。

## 一、每日灵感胶囊 (Fallback Prompts)

系统将从以下问题中随机抽取：

1. 今天发生的最让你感激的小事是什么？
2. 如果今天可以重来，你会做什么不同的选择？
3. 描述一下你此刻的能量状态。
4. 此时此刻，你最想对谁说一声谢谢？
5. 最近有什么事情让你感到焦虑？试着写下来。
6. 如果明天是世界末日，你今天最想做什么？
7. 你最近学到的一个重要道理是什么？
8. 今天为了目标，你迈出了哪一小步？
9. 描述一个让你感到宁静的瞬间。
10. 你对自己最近的表现满意吗？为什么？
11. 什么是你一直想做但不敢做的事？
12. 这一周你最自豪的成就是什么？
13. 如果不考虑金钱，你最想从事什么工作？
14. 你觉得什么样的生活才算“成功”？
15. 最近一次让你开怀大笑是因为什么？
16. 你的核心价值观是什么？
17. 你希望朋友如何形容你？
18. 有什么习惯是你想要改变的？
19. 你心目中理想的一天是怎样的？
20. 给未来的自己写一句话。
